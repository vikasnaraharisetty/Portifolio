APP Store

Government of Andhra Pradesh | 2018

AppStore an proprietary Web & Mobile App Store

Andhra Pradesh Play Store is a Government of Andhra Pradesh proprietary mobile app store and single source for all Government to citizen and Government to business apps. A web-based APP Store Developer Portal is dedicated to innovators and startups creating an opportunity to innovate.

Research and discovery

With the new branding already defined, I need to create and build a new website along with web application for developers and citizens , which need to be responsive. The first thing I did was try and figure out how the work flow should be done, and how we can we represent the data according to the users. Luckily the workflow was easliy defined using the requirements, so I had some idea of how data should be represnted. Some quick customer interviews also provided useful insights.

Designing the visuals

With the branding, workflow and data presentation defined it was time to start working on the website and web application. It was clear that we basically needed a simple one page website that outlined the service that is provided and gave potential customers an easy way to get in touch. We decided to go with a simple one-page website with both Android and IOS applications shown ready to download.

I sketched out a few quick concepts for the user centric view and then got straight into Photoshop. We decided to create a data grid that represents the application data with simple understanding. As the application need to go throught different stages to finally get published in APP Store, the major consern is to keep the track of the application during each level of validation, so we designed a custom data grid that represents the status using simple dots.

Now all the data is on track it is time for us to ForeSee to enhance the reporting with a customizable way for users to view their data. Our mission for this project was to empower our users, allowing them to see the data they wanted to see, in the way they wanted to see it. Yet, we knew it would be a strategic misstep to spin our wheels trying to recreate some of the complex business intelligence tools that already existed so we opted out for a simple clean design with bar charts. We knew it would be vital to find that right balance between simplicity and flexibility.

Ongoing optimisation

We plan on designing different version of the application to split test against each other. I think it’s important to be continuously trying new ideas to improve your conversion rate, rather than just concentrating on driving more visitors to your website. Designing the application itself was also an interesting challenge, so I will put together a separate case study detailing how we went about designing the interface in my blog when it is ready.
